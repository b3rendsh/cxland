## Hardware

[**RCBUS TN9K - MSX/Coleco compatible video adapter**](https://github.com/b3rendsh/cxland/tree/main/hardware/rcbus-vdp9k)  
 The card uses the Tang Nano 9K FPGA to emulate the TMS9918A (F18A) and V9938/V9958 VDP with 32K VRAM.  
 It can be set to MSX mode or Colecovision mode with a few dip switches and it supports the V9938 80 columns text mode.
